Ретенция полного съёмного протеза

Мы предлагаем создать индивидуальную ложку для получения функциональных оттисков на основе результатов конусно-лучевой компьютерной томограммы с контрастированием полости рта

Для получения результата необходимо провести компьютерную томограмму размером 10х15 см или близкий аналог по размерам (важно для нейронной сети) если на томографе есть режимы снижения помех от металлических протезов в полости рта или есть режим получения снимков с контрастным веществом то желательно активировать вышеозвученные режимы.

Для контрастирования полости рта рекомендуется применение препаратов на основе перорального сульфата бария (*перорального - разрешенного для приема внутрь). Препараты на основе йода могут впитываться в полости рта и отображать подповерхностные слои эпителия

Нейросеть на основе результатов КЛКТ проводит анализ рельефа слизистой оболочки и генерирует объёмную цифровую модель протезного ложа.

От протезного ложа проводится построение ручки ложки с встроенным регулятором высоты окклюзии, формируются боковые отделы с отформовками под окклюзионные плоскости совпадающие с кемперовкой либо франкфуртской горизонталью которую нейросеть "находит по костным ориентирам, а не по мягкотканным точкам как "кожанный" врач.

Рассмотрим установку и настройку серверной части на примере Ubuntu Server 18.04, затем выполним подключение к серверу из операционных систем Linux и Windows.Все действия на сервере выполняем под учетной записью обычного пользователя, в нашем случае в системе для примера создан пользователь mihail, а учетная запись root по умолчанию отключена.
